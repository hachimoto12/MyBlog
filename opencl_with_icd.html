AMD APP SDKが無いので、代わりにOpenCL Install Client Driver(ICD) LoaderでOpenCLを使ってみる

作成日 : 2019/07/07
更新日 : 2019/07/07
folder ： プログラミング
local_offer ： OpenCL,c++

AMD APP SDKが404のため。

環境

  • Windows 10
  • Radeon RX 560.
  • Visual Studio 2017 Communityインストール済み。

やり方

  1. AMDのドライバーを最新にしておく。
  2. c:/windowsにあるOpenCL.dllが新しいことを確認。
  3. OpenCL-ICD-Loaderからプロジェクトをダウンロード。
  4. OpenCL Headerからヘッダーの最新版をダウンロード
  5. 3と4.を解凍。
  6. OpenCL HeaderのCLフォルダをOpenCL-ICD-Loaderのincフォルダへコピー。
  7. OpenCL-ICD-Loaderのスクリプトを実行。
  8. OpenCLのプログラムを書く。
  9. ビルドの設定で、ライブラリをOpenCL-ICD-Loaderのbuild/OpenCL.libを指定する。
  10. ビルド
