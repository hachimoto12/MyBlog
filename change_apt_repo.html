aptのレポジトリ先を変更

作成日 : 2020/12/26
更新日 : 2020/12/26
folder ： Linux
local_offer ： Linux

インストール時に apt のデフォルトで設定されているリポジトリ先を日本内のリポジトリに変更する。
リポジトリ先として IIJ や jaist などがあるが、今回は riken にする。
次のコマンドを実行する。 

sudo sed -i.bak -e "s|http://archive.ubuntu.com/ubuntu|http://ftp.riken.jp/Linux/ubuntu|g" /etc/apt/sources.list
